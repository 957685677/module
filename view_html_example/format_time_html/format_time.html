时间的格式化

使用之前

  • 后台返回的必须是时间戳,这里引用了：

    commonJs里的dateCommon.js

  • 两种格式按要求来写

使用方法

  • 第一种使用方法：

    (new Date()).Format("yyyy-MM-dd hh:mm:ss")

支持以下时间格式：

  • 2018-04-16 18:33:42
  • 2018年04月16日 18时33分42秒
  • 2018.04.16 18:33:42

  • 第二种使用方法：

    FormatDate(new Date(),"yyyy-MM-dd hh:mm:ss")

支持以下时间格式：

  • 2018-04-16 18:33:42
  • 2018年04月16日 18时33分42秒
  • 2018.04.16 18:33:42

  • 第三种使用方法：

    ForgitDate(new Date())

支持以下时间格式：

  • 1510369871000
  • 1513051932000
  • 1487773342000
